## Getting Started: 

- Clone repository 

```bash
mkdir ~/Dev/real-estate -p
cd ~/Dev/real-estate
git clone https://github.com/Arvind-4/real-estate.git .
```  

- Install Dependencies:

```bash
cd ~/Dev/real-estate
npm i
```


- Run Server:

```bash
npm run dev
```

Open [localhost:3000](http://localhost:3000) in your favourite browser :)
